# 📱 Apps Store Management Project

## 🚀 Project Overview

This is a modern **Apps Store Management Web Application** built using React.
Users can explore different applications, view detailed information, install apps, and manage installed apps easily.

The project simulates a real-world App Store experience like Play Store or App Store.

---

## ✨ Features

* 🔍 Live Search Apps by Title
* 📊 App Details Page with Rating Distribution Chart
* 📥 Install Apps (Saved in LocalStorage)
* 🗑️ Uninstall Installed Apps
* ⭐ Sort Installed Apps by Rating and Size
* 🔔 Toast Notifications using React Hot Toast
* 📱 Fully Responsive UI (Mobile + Desktop)

---

## 🛠️ Technologies Used

* React JS
* React Router
* Tailwind CSS + DaisyUI
* Recharts (for Rating Chart)
* React Hot Toast
* LocalStorage (for Install System)

---

## 📂 Pages

* Home / Apps Page
* App Details Page
* Installed Apps Page
